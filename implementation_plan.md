# План реализации: Сайт Gala Decors

Основано на требованиях из `mission.md` и референсах из `examples-ui.md`.

## Цель
Разработать элегантный, минималистичный сайт-визитку для каталога декора с последующим размещением на GitHub Pages.

## Предлагаемая архитектура и структура
Проект будет реализован на **Vanilla HTML, CSS и JavaScript**. Это идеальное решение для легкого, быстрого и красивого статического сайта для GitHub Pages без перегрузки фреймворками.

### Дизайн и стилистика
- **Основа**: Скандинавский минимализм, чистые формы, пространство вокруг элементов.
- **Цвета**: Спокойные, пастельные и нейтральные тона (айвори, бежевый, пыльно-розовый или мягкий серый), чтобы фокус оставался на красоте самой продукции.
- **Типографика**: Элегантные Serif-шрифты для заголовков (например, Playfair Display или Lora) и чистые Sans-serif для текста (Inter/Roboto).
- **Анимации**: Микро-анимации кнопок, плавное появление элементов (Fade In / Slide Up) при прокрутке страницы.

### Файловая структура (Компоненты)

#### `index.html` (Главная)
Вводный баннер (Hero-блок), короткая презентация бренда, хиты продаж, ссылки на соцсети.

#### `catalog.html` (Каталог)
Страница с сеткой товаров. У товаров будут цены, фото (позже можно добавить видео) и красивое оформление карточек.

#### `about.html` (Обо мне)
История мастера, философия бренда Gala Decors.

#### `order.html` (Под заказ)
Условия индивидуального заказа, контактная информация для связи.

#### `style.css` 
Единая таблица стилей (с использованием CSS переменных для цветов и шрифтов).

#### `script.js`
Скрипт для работы мобильного меню и инициализации элегантных анимаций при прокрутке.

## План тестирования и проверки
### Как будем проверять:
1. Откроем локально в браузере и проверим переходы по страницам.
2. Протестируем адаптивность на узких экранах (мобильный вид).
3. Убедимся, что ссылки на контактную информацию (телефон, email, Instagram, Facebook) правильные.
4. Проверим плавность анимаций.
