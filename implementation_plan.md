## Фаза 3: Логика загрузки и обработка фото (Завершено)
- [x] **Первичная обработка массива фото:** Написан Python-скрипт, преобразовавший 50+ тяжелых скриншотов из `raw_uploads` (до 50 МБ) в оптимизированный JPEG-формат (1000px) с деликатным повышением насыщенности красок (+15%).
- [x] **Алгоритм классификации:** Все файлы рассортированы по 13 категориям-папкам на основе их префикса (`cat`, `dog`, `wreath`, `frame`, и т.д.).
- [x] **Динамический каталог (JS):** Файл `script.js` парсит автоматически созданные `*.txt` файлы и генерирует сетку товаров на основе фильтра. Никаких строк ручного HTML-кода для каждого товара больше вводить не нужно.
- [x] **Инкрементная (Добавочная) загрузка:** Скрипт дополнен возможностью "доливать" новые фото к старым папкам — подхватывать последний старший номер в папке, добавлять новые товары (например, `4` и `5`) и дописывать строки в конец текстового прайса, не ломая то, что было.
- [x] **Блок Фавориты (favorites):** Выведен отдельно для Главной страницы `index.html`.
